# CareConnect: Healthcare NGO Support Platform

Welcome to the **CareConnect** repository! This web application was built as an internship assignment to provide a comprehensive, digital-first support platform for healthcare Non-Governmental Organizations (NGOs). It streamlines patient support, volunteer registration, and general inquiries, while introducing an automated FAQ system to handle common questions.

## 🚀 Live Demo
[https://health-care-seven-xi.vercel.app/](https://health-care-seven-xi.vercel.app/)

## 🛠️ Tech Stack
This project is built using modern web development technologies to ensure performance, responsiveness, and a smooth user experience:
- **React.js**: Front-end library for building component-driven user interfaces.
- **Vite**: Next-generation frontend tooling for extremely fast development and build times.
- **Tailwind CSS**: Utility-first CSS framework for rapid, responsive, and highly customizable UI styling.
- **React Router**: Standard routing library for React to handle seamless single-page application navigation.

## ✨ Key Features
- **Patient Support Form**: A dedicated, easy-to-use channel for patients to request medical assistance, report issues, or seek guidance.
- **Volunteer Registration Form**: A structured application form for individuals looking to contribute their time and skills to the NGO.
- **Contact Form**: A general inquiry system for partners, donors, and other stakeholders.
- **AI FAQ Chatbot (MediBot)**: A floating interactive chatbot that instantly answers common healthcare and NGO-related questions directly from the UI.

## 🤖 AI/Automation Idea: The Future of MediBot
Currently, **MediBot** operates using a predefined logic system—it matches user inputs against a static dictionary of Frequently Asked Questions to provide instant responses safely. 

**Future Upgrade Path:**
To make the application significantly more intelligent, MediBot can be integrated with a real Large Language Model (LLM) such as the **OpenAI API**. 
- **Smarter Responses**: An LLM could understand natural language context, seamlessly handling misspellings, complex phrasing, and detailed follow-up questions organically.
- **Dynamic Knowledge Base**: Instead of hard-coded FAQs, the bot could utilize RAG (Retrieval-Augmented Generation) connected to the NGO's actual documentation, medical guidelines, and past support tickets to provide highly accurate, up-to-date, and context-aware assistance.

## 🌍 NGO Use Case: Real-world Impact
For a real healthcare NGO, managing communication can be overwhelming. This platform solves several critical operational bottlenecks:
1. **Centralized Intake**: Instead of relying on scattered emails or phone calls, patient needs and volunteer applications are systematically routed through structured forms.
2. **24/7 Support**: MediBot ensures immediate answers to standard questions (e.g., "What are your operating hours?", "How can I volunteer?"), reducing the administrative load on human staff and ensuring users feel heard immediately.
3. **Scalability**: The modern, responsive tech stack ensures the platform can handle growing traffic during health campaigns without performance degradation.

## 💻 How to Run Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sahil1925m/HealthCare.git
   ```
2. Navigate to the project directory:
   ```bash
   cd healthcare-ngo-app
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173` (or the port specified in your terminal).
